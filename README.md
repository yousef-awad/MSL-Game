# MSL Sign Language Game

A real-time sign language challenge game using AI-powered gesture recognition for Malaysian Sign Language (MSL).

## How to Play

1. Click "Start Game" to begin
2. You'll be shown a random word
3. You have 15 seconds to perform the sign
4. The AI will recognize your sign in real-time
5. Score points for correct signs and build up your streak!

## Features

- Real-time gesture recognition using webcam
- 15-second challenge timer
- Score tracking (correct, wrong, streak)
- Immediate feedback on predictions
- Clean, modern UI

## Technology

- Frontend: HTML, CSS, JavaScript
- Backend API: Malaysian Sign Language Recognition Model
- Deployment: GitHub Pages
- CI/CD: GitHub Actions
